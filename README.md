# Python Banking Management System

A complete Banking Management System developed using **Python** and **Tkinter GUI** with modern interface design and Object-Oriented Programming concepts.

This project allows admins to manage customers, perform banking operations, generate reports, and securely handle transactions with persistent JSON data storage.

---

# Features

## Admin Features

- Secure Admin Login System
- Full Admin Rights Management
- Create New Customer Accounts
- Update Customer Information
- Delete Customer Accounts
- Update Admin Profile
- Search Customers
- View All Customers
- Management Reporting System

---

## Banking Features

- Deposit Money
- Withdraw Money
- Transfer Money Between Accounts
- Balance Checking
- Account Validation
- Insufficient Balance Handling
- Invalid Account Handling
- Secure Transaction Flow

---

## Account Types

### 1. Normal Account

Standard customer account.

### 2. Savings Account

Supports yearly interest calculation.

### 3. Advanced Account

Supports overdraft limit functionality.
